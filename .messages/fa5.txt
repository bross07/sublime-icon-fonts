Font Awesome v5
===============

The final version of Font Awsome 5.0.0 has been released today! So, this
package has been updated to support all of its icons. If you rely on using
Font Awsome 4.x, please install the “Icon Fonts (Legacy)” package.

References:

- https://fontawesome.com/
- https://packagecontrol.io/packages/Icon%20Fonts%20%28Legacy%29
